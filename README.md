# GitHub Basics Workshop

This repository is created for learning Git and GitHub from scratch.

## What you will learn
- What Git and GitHub are
- How to create a repository
- How to make commits
- How to work with branches
- How to open Pull Requests

## Project structure
- README.md → Project description
- notes/ → Learning notes
- tasks/ → Practice tasks

## How to start
1. Clone the repository
2. Follow the tasks step by step
3. Practice Git commands
